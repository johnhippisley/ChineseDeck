# ChineseDeck

A media reach Chinese Vocabulary for Anki

---

## Features

  - Pinyin
  - Audio
  - Example sentences
  - Images
  - Stroke order animations
  - 5,716 organized vocabuarly cards

## Tags
Cards are organized into various vocabularly lists, including the chapters from Chinese Link Part 1 & 2 and All Things Considered. 
They also are organized into HSK1 - HSK6 and a few other categories.
<p align="center">
  <img src="media/tags.png" alt="Tag structure" width="500">
</p>

I recommend you start with all cards suspended, and then gradually unsuspend cards as you go.
